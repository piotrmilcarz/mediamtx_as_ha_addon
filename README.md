# MediaMTX Home Assistant Add-on Repository

Home Assistant add-on repository for [MediaMTX](https://github.com/bluenviron/mediamtx) — a real-time media server supporting RTSP, RTMP, HLS, WebRTC and SRT.

## Installation

1. Open **Settings → Add-ons → Add-on Store** in Home Assistant.
2. Click the three-dot menu (⋮) in the top-right corner and choose **Repositories**.
3. Add the following URL:
   ```
   https://github.com/piotrmilcarz/mediamtx_as_ha_addon
   ```
4. Find **MediaMTX** in the store and click **Install**.
